About Gigs Kenya

A smarter freelance marketplace for Kenyan talent and hiring teams.

Gigs Kenya is designed to help skilled workers and serious employers connect inside a practical, professional environment. Beyond listings, the platform supports messaging, application review, workspaces, reviews, payout flows, top workers visibility, and useful account tools that make the whole job cycle easier to manage.

Mission

Make digital work opportunities easier to access, easier to manage, and more trustworthy for people hiring and working in Kenya.

Focus

Professional user experience, local relevance, clear job workflows, practical payment handling, and strong visibility for good workers.

Approach

Build credibility through reviews, structured applications, competency signals, work history, and cleaner communication between both sides.

How It Works

From posting to payout, the workflow stays clear.

Quick summary

Employers post and review. Workers apply and deliver. Both collaborate in a shared workspace, then completion and reviews strengthen platform trust.

1

Create or sign in

Join as a worker or employer, complete your profile, and access the dashboard tools that fit your role.

2

Post or apply

Employers publish gigs. Workers browse, filter, save jobs, answer custom questions, and submit pitches.

3

Collaborate in workspace

Use milestones, file vault, time logs, quick actions, submissions, approval states, and project messaging.

4

Complete, review, grow

After completion, reviews update trust signals, invoices stay available, and strong workers rise through ratings and visibility.

Worker journey

Browse jobs, filter by budget, and bookmark promising opportunities.

Apply with a pitch, rate, delivery timeline, and any competency responses requested.

Work inside the shared workspace, communicate updates, log time, store files, and submit deliverables.

Earn reviews, improve ranking, request payouts, and use your referral code to grow earnings.

Employer journey

Post a detailed gig with category, budget, deadline and job questions if needed.

Review applications or directly hire a strong worker from their public profile.

Track delivery progress, review submitted work, request changes, approve work, and keep a clean project history.

Leave reviews and build an employer reputation that attracts better applicants over time.

Trust & Safety

Helpful safeguards for better marketplace behavior.

Gigs Kenya already includes practical product features that encourage better behavior: profile transparency, public reviews, competency checks, approval flows, activity logs, file history, dispute escalation, and notification records.

Safer collaboration habits

  • Use detailed job descriptions and written expectations.
  • Keep communication inside platform messages and workspace updates.
  • Review profiles, skill tags, ratings and recent reviews before hiring.
  • Use milestones, approvals and disputes when work becomes unclear.

Useful account tools

  • Notification history for important events
  • Activity logs inside workspaces
  • Review submissions after gig completion
  • Public reporting options on visible profiles
